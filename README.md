Personalized Study Buddy is an AI-powered assistant built using Python and LangChain that helps users learn any topic. 
It generates simple explanations and multiple-choice quizzes based on the topic entered by the user using the Groq LLM.

How to Run
1.Clone the repository
git clone https://github.com/vijayadurgagotla-lab/PersonalizedStudyBuddy.git

2.Go to the project folder
cd PersonalizedStudyBuddy

3.Install dependencies
pip install langchain langchain-groq

4.Set your Groq API key
set GROQ_API_KEY=your_api_key

5.Run the program
python PersonalizedStudyBuddy.py

Your Personalized Study Buddy will start and ask for a topic to learn.
